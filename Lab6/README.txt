Invoking program details
------------------------------------------------------------------------------
Invoke program: python rrt.py [obstacle file] [start goal file]
Example: python rrt.py obstacles.txt start_goal.txt

Two coordinate files are needed to supply obstacle and goal point information. The format is as specified in the lab instructions.
